# 🚦 Arduino Traffic Light Control (3 LEDs)

This project simulates a basic traffic light system using an Arduino and three LEDs (Red, Yellow, Green). Each LED is connected to a digital pin with a current-limiting resistor.

---

## 🚀 Features
- Simulates real-world traffic light sequence
- Uses three LEDs: Red, Yellow, Green
- Simple timing control using delay()
- Beginner-friendly Arduino project

---

## 🧰 Components Required
- Arduino board (Uno/Nano/etc.)
- 1 × Red LED
- 1 × Yellow LED
- 1 × Green LED
- 3 × 220Ω resistors
- Jumper wires
- Breadboard (optional)
